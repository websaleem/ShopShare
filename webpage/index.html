How it works

Pick a tab below to add items. Then watch the per-person split fill in automatically.

AI receipt import

Upload a JPG, PNG, or PDF of the receipt. The Textract + Bedrock pipeline extracts each line and price; you review before they're added.

Manual entry

Type item, price, and the person it belongs to. Best for quick splits where you remember everything.

S3 receipt vault

Optionally store the original receipt in your private S3 bucket. The browser PUTs it directly through a presigned URL — never via the Lambda.

Tax & discount aware

Set service tax and discount percentages once; ShopShare prorates them across each person in proportion to their subtotal.

No client API keys

Your requests securely route through CloudFront without exposing any API keys in the browser.

Serverless on AWS

Static frontend on S3 + CloudFront, a Lambda Function URL for the AI call and presigned-URL signing. Runs in the AWS free tier.
